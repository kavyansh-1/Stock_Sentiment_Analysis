# Stock Sentiment Analysis (News Headlines)

## Pipeline

```
[News Headlines]
        ↓
[Text Preprocessing]
(lowercase, clean, tokenize)
        ↓
[Feature Extraction]
(TF-IDF / CountVectorizer)
        ↓
[Train/Test Split]
        ↓
[Model Training]
(Logistic Regression / Naive Bayes)
        ↓
[Evaluation]
(Accuracy, F1-score)
        ↓
[Prediction]
(New headline → Sentiment)
```
